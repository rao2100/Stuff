## Tmux:
``` bash
$ cp .tmux ~/
$ git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
$ tmux source ~/.tmux.conf

```
Run tmux and then Press prefix + I (capital i, as in Install) to fetch the plugins.
